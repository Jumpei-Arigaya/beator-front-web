## インストール

### 事前準備

node.js をローカルにインストール

### ローカルに開発環境を構築

#### リポジトリを clone

```
git clone git@github.com:Jumpei-Arigaya/beator-front-web.git
```

#### プロジェクトルートに移動

```
cd beator-front-web
```

#### npm の依存パッケージをインストール

```
npm install
```

### フロントエンド

#### 開発サーバを起動

```
npm run dev
```

#### ブラウザからトップページにアクセス

http://localhost:3000

#### 開発サーバを停止

ctrl + c

### Storybook

#### Storybook のサーバを起動

```
npm run storybook
```

#### ブラウザから Storybook のカタログにアクセス

http://localhost:6006

#### Storybook のサーバを停止

ctrl + c
